HTML, CSS and Java Script

HTML

HTML is Hyper Text Markup Language. It is the language used to give structure to web pages. It follows DOM - Document object model. HTML program flows from top to bottom.

CSS

CSS is the abbreviation for Cascaded Style Sheets. It gives style to web pages. It controls fonts, colors, alignment, styling, images etc. It is a very powerfool tool in web development framework. The same HTML structure can be displayed in multiple formats using different CSS

Java Script

Java Script gives functionality to web pages. It helps in enhancing user experience.
